## Project Description
Team 11 Store is a full stack e-commerce product module built with Node.js, Express.js, and MongoDB Atlas. The application supports CRUD operations for products and provides a getAll endpoint to display all products in the store.

## Features
- Create a product
- Read all products
- Read a product by ID
- Update a product
- Delete a product
- Automated testing using Jest and Supertest
- automate.js to run test scripts

## Technologies Used
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Jest
- Supertest
- Thunder Client

## Product Fields
Each product includes:
- storeId
- storeName
- productId
- productName
- price
